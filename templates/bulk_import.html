Instruções de Preenchimento
Campos Obrigatórios:
  • Título: Mínimo 5 caracteres (Ex: Compra de Laptops Dell)
  • Descrição: Mínimo 10 caracteres (Detalhe o que está sendo pedido)
  • Status: Use exatamente: orcamento, fase_compra, a_caminho, ou finalizado
  • Prioridade: Use: baixa, media, alta, ou urgente
  • Impacto: Use: baixo, medio, ou alto
  • Classe: Use: ensino ou manutencao
  • Categoria: Use: material, servico ou ambos separados por vírgula material,servico
Campos Opcionais:
  • Valor Estimado/Final: Use números com ponto (Ex: 1500.50). Não use R$ ou vírgula para decimais.
  • Responsável: Digite o NOME COMPLETO do usuário como está cadastrado no sistema.
  • Observações: Texto livre para anotações extras.
  • Data Solicitação: Formato DD/MM/AAAA. Se deixar vazio, será usada a data de hoje.
Atenção Importante:

O sistema valida os nomes dos responsáveis. Se o nome não coincidir exatamente com um usuário ativo, o pedido será criado sem responsável atribuído.
